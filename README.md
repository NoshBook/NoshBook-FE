# NoshBook

Find and customize recipes, get your weekly meal plan ready, and build your shopping list all in one app.

## Tech Stack

- Node.js
- Express
- Postgres
- React
- Typescript

Backend repo: https://github.com/NoshBook/NoshBook-BE
